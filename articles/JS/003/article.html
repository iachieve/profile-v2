What is the difference between Slice and Splice in JavaScript?

What Array.splice method does and how can we use it?

  • The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.
  • Splice method returns the removed items.
  • Splice method changes the original array.

Array.splice( index, remove_count, item_list ) The splice method takes three parameters: index: what position to add/remove items. howMany/deleteCount: the number of items to be removed. If set to 0, no items will be removed. item1...itemX: The items to be added(just added) to the entry.

Let's see Array.slice now, then compare both methods

  • The slice() method returns the selected elements in an array, as a new array object.
  • The slice() method selects the elements starting at the given start argument, and ends at, but does not include, the given end argument.
  • slice does not alter the original array. It returns a shallow copy of elements from the original array.
