What is Currying in JavaScript?

In simple English: Currying is when a function doesn't take all of it's arguments upfront.
Instead, it wants you to give it the first argument, and then the function returns another function.
Which you are supposed to call with the second argument.
Which in turn will return a new function.
Which you are supposed to call with the third argument.
Until the the last function in the chain, and then the last function in the chain will return the value that you really want.
