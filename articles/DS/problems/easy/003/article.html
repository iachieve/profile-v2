JavaScript Arrays Q and A

Given an array of integers, find the pair of adjacent elements that has the largest product and return that product.

Example For inputArray = [3, 6, -2, -5, 7, 3], the output should be adjacentElementsProduct(inputArray) = 21. 7 and 3 produce the largest product.

Solution 1: Using Two Pointer Technique

In this problem we need adjacent elements in the array [3, 6, -2, -5, 7, 3] In other word, we need to compare the multiplication result from each adjacent pair pairs in this example are [3, 6], [6, 2] , [-2, -5], [-5, 7], [7, 3] and results respectively are 18, 12, 10, -35, 21 and the right solution obviously in the last pair [7, 3];

we can solve this problem without using extra memory, but we have to iterate through the given array once at least
